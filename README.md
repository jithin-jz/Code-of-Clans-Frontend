# Code of Clans

A gaming-themed profile application built with React, Vite, and Django + Supabase.

## Features
- **Gaming Profile**: Customize your avatar, banner, and bio.
- **Social**: Follow other users.
- **Supabase Storage**: Upload high-quality images.
- **Secure**: OAuth authentication (GitHub, Google, Discord).

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the development server:
    ```bash
    npm run dev
    ```
